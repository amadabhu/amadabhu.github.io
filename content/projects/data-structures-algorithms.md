23 single-file implementations spanning **CLRS chapters 6–26**: linked structures,
hash tables, BSTs and red-black trees, B-trees, Fibonacci heaps, van Emde Boas
trees, sorting (heap / quick / counting / bubble), DP and greedy, disjoint-set
forests, MST (Kruskal), shortest paths (Dijkstra and Floyd-Warshall), and
Edmonds-Karp max flow.

Every file is self-contained, uses only the standard library, and runs as a
script to print a small demo. The README has a complexity table sorted by
Big-O plus per-file ASCII diagrams.
